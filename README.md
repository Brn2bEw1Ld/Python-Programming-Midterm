# Python Programming Midterm – Mad Lib Project

This is a group midterm project where we created three separate Mad Lib programs that together form a full story.

## Project Structure

Python-Programming-Midterm/
│
├── student1_zacks_games.py
├── student2_kaylas_games.py
└── student3_zacks_kaylas_games.py

## Description

- **Student 1 – Favorite Game Adventure**
  - Builds a Mad Lib based on Student 1's favorite video game

- **Student 2 – Favorite Game Challenge**
  - Builds a Mad Lib based on Student 2's favorite video game

- **Student 3 – Ultimate Game Crossover**
  - Combines Student 1 and Student 2's favorite games into one crossover Mad Lib

Each program runs independently and allows the user to input words to generate a unique story.
