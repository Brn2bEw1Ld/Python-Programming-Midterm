# Python Programming Midterm – Mad Lib Project

This is a group midterm project where we created three separate Mad Lib programs that together form a full story.

## Project Structure

Python-Programming-Midterm/
│
├── student1_morning_madness.py
├── student2_crazy_commute.py
└── student3_wild_ending.py

## Description

- **Student 1 – Morning Madness**
  - Introduces the start of the day

- **Student 2 – Crazy Commute**
  - Shows the journey and unexpected events

- **Student 3 – Wild Ending**
  - Concludes the story with a twist

Each program runs independently and allows the user to input words to generate a unique story.
